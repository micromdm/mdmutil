# MDM Util

[![CI/CD](https://github.com/micromdm/mdmutil/actions/workflows/on-push-pr.yml/badge.svg)](https://github.com/micromdm/mdmutil/actions/workflows/on-push-pr.yml) [![Go Reference](https://pkg.go.dev/badge/github.com/micromdm/mdmutil.svg)](https://pkg.go.dev/github.com/micromdm/mdmutil)

MDM Util contains tools and a Go library powering them for working with Apple MDM.

## Getting started & Documentation

- [Operations Guide](docs/operations-guide.md)  
A brief overview of the `mdmutil` command line tool.

## Getting the latest version

* Release `.zip` files containing the project should be attached to every [GitHub release](https://github.com/micromdm/mdmutil/releases).
  * Release zips are also [published](https://github.com/micromdm/mdmutil/actions) for every `main` branch commit.
